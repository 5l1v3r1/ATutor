5.0 Enrollment

The student list for a particular course determines which of your students have access to the course content. ATutor has the facility to allow instructors to create, import and export student lists.

To administer the users in your course, you need to be logged in as the instructor and have selected the Enrollment option from the Manage tab.

5.1 Creating, Importing, and Exporting Course Lists

5.1.1 Creating a Course List

You have the option of manually generating the student list. This option is useful if you have a small number of students to add to a course, but if you have many students then the Import may be more beneficial. Input students manually into your course, click the Create Course List option.

5.1.2 Creating a Course Enrollment List for Import

To import a list as a file from your local system to ATutor, the enrollment list should follow the format specified by your administrator. By default, the format for the list is "firstname", "lastname", "email" with one student per line, but your administrator may have a different format.

Such lists can be easily generated by programs like a spreadsheet, or a database. Once you have the list as a file, you are ready to import it into ATutor.

5.1.3 Importing Course Enrollment Lists

To import a list of students into your course, you must first have a file on your local computer containing all the necessary information (see you have a course enrollment list as a file on your local system, you are ready to import it into ATutor. To import a course list, select Import Course List link and then Browse. Finally select Import Course List button to upload the list.

When importing an enrollment list, ATutor will automatically generate user names based on the firstnames and lastnames. You have the option of seperating the username using an underscore or a period. (i.e. John_Smith, or John.Smith).

5.1.4 Exporting Course Enrollment Lists

A course enrollment list can be exported from ATutor which is useful as a backup, or for use in other courses (through importing). You can select subsets of students to export (enrolled students, un-enrolled students, and alumni). Selecting Export will begin a download in your browser. The exported list is in comma-seperated format.

5.2 Groups, Roles & Privleges

5.2.1 Groups

Students can be placed into groups. This would allow the instructor to give different tests to different groups.

5.2.2 Roles & Privleges

Each student in your course can be assigned a role and special administrative privleges. This allows some (or all) of your students to perform actions like manage the glossary, or administer the forums.

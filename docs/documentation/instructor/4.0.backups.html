4.0 Backups

ATutor offers the facility to create backups of your course and restore these copies at any time. This is very useful for saving revisions or give extra security for your course.

To manage backups, you must be logged in as an instructor and have selected a course to backup. Under the Manage tab, choose the Backup option.

4.1 Creating and Restoring a Backup

To create a backup of the current course, select the Create link from the options. Enter a description if needed and click Create to make a backup of the course. The backup will reside on the ATutor system and can be optionally downloaded to your local computer.

Once a backup has been created, you can restore it by clicking the Backups link, and then clicking the Restore button after selecting a backup to restore. Upon clicking the Restore button, you are given options for what material you want restored.

4.2 Downloading and Uploading a Backup

Backups can be downloaded to your local computer by selecting the backup and clicking the Download button. Your browser would then start a download of the backup.

The backup is compressed in a ZIP file. Unlike the Content Package Export feature in ATutor, the backup is not SCORM or IMS compliant.

You can also upload backups from your local system to ATutor. Simply click on the Upload link, fill in the necessary information, and click Upload to finish. The backup will then appear in the list of backups.

Downloading and uploading backups also allows you to exchange content from one course to another.

4.3 Editing and Deleting a Backup

Selecting a backup and clicking the Delete button will delete that backup. You will be asked to confirm the action before actual deletion occurs. Once a backup is deleted, it can't be restored.

Selecting a backup and clicking the Edit button will allow you to edit the description of that backup.

7. Tests & Surveys

ATutor has the functionality to create tests and surveys to be administered to students enrolled in your course. There are many different options for delivering a test such as randomized questions, and group-specific tests.

Tests & Surveys can be found under the Manage tab when logged in as a course's instructor.

7.1 Creating a Test or Survey

Select the Create Test/Survey link to begin creating a test or survey. There are a few options available and they are defined as follows:

Title
The title of the test or survey.
Attempts Allowed

You can limit the number of attempts a student is allowed for the test. If a student uses up their allotted attempts, then that test/survey is no longer available for them to take.

Each attempt is recorded in the test/survey statistics, and in the student's account.

Anonymous
If this option is set to "Yes", then the instructor will not be able to see the names of those students who have answered the test/survey.
Release Results
This option controls when the results of a student's test is shown to them. The "Once quiz has been submitted" option will grade all possible questions, except open ended questions, and give the results immediately to the student.
Randomize Qustions
This option will change the order which questions appear on a test/survey if enabled. You can specify the number of randomized questions that will appear in the test. If this number is equal to the total number of questions in the test, then the effect of randomizing questions is nullified.

4.1 Backups

A course backup includes all available material as an archive in a format specific to ATutor. Backups are forwards compatible with future versions of ATutor but are not backwards compatible with previous versions of ATutor. Once a backup is created it can be downloaded for safe-keeping, imported into another ATutor installation, used as the basis for a newly created course, and available in the originating course's Backup Manager.

4.1.1 Creating Backups

The number of backups a single course can keep on the server is defined by the System Preferences Course Backups option.

Administrators can create backups for any course, while instructors can only create backups of courses they own.

4.1.2 Restoring Backups

Restoring a backup as administrator is the same as when restoring a backup as an instructor, with the added option of being able to select which course the backup should be restored into.

For details additional details on restoring a backup into a course see the Backup Manager's Restoring Backups section in the Instructor Documentation.
